# COGNITIVO_045

Minijuego para el item "Imita el dibujo de una cara".

## Flujo

- Muestra una cara modelo en SVG.
- El nino copia la cara en un canvas con mouse, dedo o lapiz tactil.
- El boton "Termine mi dibujo" registra la actividad como completada.
- La validacion queda marcada para revision adulta/profesional.

## Evidencia/metadata

El resultado incluye cantidad de trazos, duracion y tamano del canvas en `respuesta_usuario`.
